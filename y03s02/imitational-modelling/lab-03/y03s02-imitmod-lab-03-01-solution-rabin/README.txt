Как пользоваться:
1. Открываем командную строку в папке. (Shift + Правая Клавиша Мыши → Открыть окно команд).

2. В появившемся окне запускаем скрипт set_env.bat:
..\y03s02-imitmod-lab-NN> set_env.bat

Командная строка изменится (появится префикс «(venv)»):
(venv) ..\y03s02-imitmod-lab-NN>

3. Запускаем программу:
(venv) ..\y03s02-imitmod-lab-NN> python <название_файла_программы>.py

4. Программа должна вывести нужные данные в окно командной строки и открыть окно с графиком, если это нужно в задании.
