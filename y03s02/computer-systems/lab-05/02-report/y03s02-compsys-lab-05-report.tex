	\section{Хід роботи}
		Вихідними даними для лабораторної роботи є~арифметичний вираз. Для варіанту~№~8 він виглядає так:
		\begin{IEEEeqnarray}{rCl}
			A \div B + (C + D \times E) \times (G + K \div L \times M).
		\end{IEEEeqnarray}
