Логічні елементи

Я захоплююсь модою і модною фотографією (англ. fashion photography). Мода в одязі — це зміна форм і зразків одягу, яка відбувається протягом відносно коротких проміжків часу. Модна фотографія — це жанр фотографії, призначений для зйомки одягу, прикрас, аксесуарів та інших предметів моди.

Під час зйомок часто стають у нагоді метадані зображення. Щоб їх прочитати, використовують утиліту exiftool. Її можна знайти за допомогою комбінації клавіш Win + S або Cmd + Space. При запуску програма виводить таке повідомлення ExifTool Version Number: 7.63
File Name: 01.cr2
...
 Також exiftool надає інтерфейс для програмного доступу. Тоді інтерфейс доступу позначають як exiftool.

Ось приклад використання зазначеного шрифта (Sitka Text): The five boxing wizards jump quickly.

Фізичні елементи

Курсив (<i></i>)
Нахилений текст для виділення.
Напівжирний (<b></b>)
Товстий текст для виділення.
Моноширинний (<tt></tt>)
Усі символи однакової ширини. Для таблиць або початкового коду.
Підкреслений текст (<u></u>)
Використовують для виділення за традицією друкарських машинок.
Закреслений (<strike></strike>)
Адаптовано з письма.
Закреслений (<s></s>, коротка форма)
Більш компактна форма.
Дрібний текст (<small></small>)
Приховані умови.
Великий текст (<big></big>)
Гучні обіцянки.
