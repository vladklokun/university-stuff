      \subsection{Створення форм~\textenglish{\allcaps{HTML}}}
        Щоб~виконати наступне завдання, необхідно створити форму, яка~відповідатиме таким вимогам:
        \begin{enumerate}
          \item Список перемикачів: синій, бузковий, блакитний, зелений, причому слова повинні відображатися відповідним кольором.
          \item Список прапорців повинен розміщуватися в~центрі сторінки і~містити назви восьми геометричних фігур.
          \item Дані про~себе користувач повинен вводити в~текстові поля («Прізвище», «Ім’я»), а~кафедра (\allcaps{КСУ} або~\allcaps{КСМ}) і~номер групи (320–326) вибираються за~допомогою списків.
          \item При~натисканні кнопки «Відправити» повинна запускатися програма для~відправки такого повідомлення:
            \begin{codegeneric}
              сvet = синій
              figura = еліпс
              figura = прямокутник
              fam = Шевченко
              imya = Максим
              fakultet = КСУ
              gruppa = 325
            \end{codegeneric}
            \item Після натискання кнопки «Очистити поля форми» всі~поля форми мають набути початкового вигляду.
            \item В~кінці сторінки створіть елемент~\minttext{textarea} із~заданим текстом.
        \end{enumerate}
        Отже, створюємо веб-сторінку за~заданими вимогами~(лістинг~\ref{lst:03-form}). В~результаті отримали бажану веб-сторінку з~необхідною~\textenglish{\allcaps{HTML}}-формою~(рис.~\ref{fig:03-form-res}).
