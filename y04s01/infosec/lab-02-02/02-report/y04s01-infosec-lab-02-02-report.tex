	\section{Мета роботи}
		Ознайомлення з~програмними бібліотеками криптографічних функцій.

	\section{Завдання роботи}
		Використовуючи можливості криптографічних програмних бібліотек, створити програмний додаток, що~виконує прості криптографічні операції.

	\section{Хід~роботи}
		Щоб~виконати завдання лабораторної роботи, використаємо мову програмування~\textenglish{Python} і~модуль~\modulename{cryptography}. Перш за~все, встановимо необхідний модуль. Для~цього створюємо віртуальне середовище за~допомогою такої команди:
		\begin{codegeneric}
			python -m venv venv
		\end{codegeneric}
		В~результаті з'явиться віртуальне середовище під~назвою «\textenglish{venv}». Тепер активуємо його за~допомогою такої команди для~операційної системи~\textenglish{Windows}:
		\begin{codegeneric}
			venv\Scripts\activate.bat
		\end{codegeneric}
		або~її~аналога для~\textenglish{\allcaps{GNU}/Linux}:
		\begin{codegeneric}
			source venv\bin\activate
		\end{codegeneric}
		Тепер, знаходячись у~віртуальному середовищі, встановлюємо необхідний модуль за~допомогою такої команди:
		\begin{codegeneric}
			pip install cryptography
		\end{codegeneric}
		Завершивши виконання, команда встановить необхідний модуль. Коли модуль встановлений, розробляємо програмний додаток, який обчислить значення хеш-суми за~допомогою алгоритму~\textenglish{\allcaps{SHA-256}}, а~також зашифрує і~розшифрує повідомлення за~допомогою автентифікованого шифрування із~додатковими даними за~алгоритмом~\textenglish{ChaCha20Poly1305}~(лістинг~\ref{lst:source-code}, \ref{lst:requirements}).

		Запускаємо розроблений додаток, вводимо текст і~спостерігаємо результат~(рис.~\ref{fig:res}).

		\begin{figure}[!htbp]
			\begin{subfigure}[b]{12\gridunitwidth}
				\includegraphics[height = 8 \baselineskip]{./assets/00.png}
				\caption{}
				\label{subfig:res}
			\end{subfigure}
			\begin{subfigure}[b]{12\gridunitwidth}
				\includegraphics[height = 8 \baselineskip]{./assets/01.png}
				\caption{}
				\label{subfig:reference}
			\end{subfigure}
			\caption{Результати обчислення хеш-сум, шифрування і~розшифрування: \subref{subfig:res}~— розробленого програмного додатку, \subref{subfig:reference}~— модуля~\modulename{hashlib}}
			\label{fig:res}
		\end{figure}

		Як~бачимо, розроблений програмний додаток працює справно і~дає~правдоподібний результат. Порівнюємо хеш-суму, отриману в~розробленому додатку, із~хеш-сумою, отриманою від~реалізації в~стандартній бібліотеці~\modulename{hashlib} мови програмування~\textenglish{Python} і~бачимо, що~значення співпадають, отже алгоритм обчислення хеш-суми реалізований правильно.
